My Skills

私は、2025年6月頃からこのホームページ作成についての勉強を日々重ねてきました。
その中で現在身に付けたスキルは３つあります！

  • 1.　HTML

    これは、「見える部分の基礎」を作る技術です。
    基礎となるWebページの骨組みを作るためには欠かせないものです。

  • 2.　CSS

    これは、「デザインやレイアウト」を作る技術です。
    自分好みにWebページの見た目を整えるために必要となります。

  • 3.　JavaScript

    これは、「操作に応じて動く仕組み」を作る技術です。
    自由自在にWebページを動かすためのプログラミング言語です。

これらを使用し、ご期待に応えられるものを作成させていただきます！
